Instruções para Resolver a Tela Branca:

1. Limpar Cache do Navegador

Pressione Cmd + Shift + R (Mac) ou Ctrl + Shift + R (Windows/Linux) para fazer um hard refresh.

2. Abrir o Console do Navegador

Pressione Cmd + Option + J (Mac) ou F12 (Windows/Linux) e verifique se há erros na aba Console.

3. Verificar a Aba Network

Na aba Network do DevTools, verifique se todos os arquivos estão carregando com status 200.

4. Testar em Modo Anônimo

Abra uma janela anônima/privada e teste a aplicação.

5. Acessar a Aplicação

Acesse: http://localhost:8080/
